# 光子治疗计划系统 (MOZI-PHOTON)

## 简介
基于 MOZI 4.0 平台的光子放射治疗计划系统。

## 功能模块
- 患者管理
- 图像处理
- 靶区勾画
- 计划设计
- 计划优化
- 计划评估
- 计划QA

## 光子特有功能（后续迭代）
- IMRT / VMAT 计划设计
- MLC（多叶准直器）与控制点管理
- 光子剂量计算/评估相关工具（占位）

## 使用方法
直接打开 `index.html` 即可使用光子治疗计划系统。

## 依赖
- 共享组件库：`../shared/components/`
- 共享样式：`../shared/styles/`
- 共享资源：`../shared/assets/`

## 说明
本系统是放射治疗计划系统的高保真原型示例，用于开发参考。
