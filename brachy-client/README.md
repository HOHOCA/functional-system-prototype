# 后装治疗计划系统 (MOZI-BRACHY)

## 简介
基于 MOZI 4.0 平台的后装（近距离）放射治疗计划系统。

## 功能模块
- 患者管理
- 图像处理
- 靶区勾画
- 施源器放置
- 剂量优化
- 计划评估

## 后装特有功能
- 施源器管理（针道、球囊、模板等）
- 源活度配置
- 驻留时间优化
- 驻留位置优化
- 分次计划
- 累积剂量评估

## 使用方法
直接打开 `index.html` 即可使用后装治疗计划系统。

## 依赖
- 共享组件库：`../shared/components/`
- 共享样式：`../shared/styles/`
- 共享资源：`../shared/assets/`

## 开发说明
- 本系统是放射治疗计划系统的高保真原型示例，用于开发参考
- 当前基于质子系统框架构建，后续将根据后装治疗的特殊需求进行定制化开发
