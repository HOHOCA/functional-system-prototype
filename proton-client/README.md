# 质子治疗计划系统 (MOZI-PROTON)

## 简介
基于 MOZI 4.0 平台的质子放射治疗计划系统。

## 功能模块
- 患者管理
- 图像处理
- 靶区勾画
- 计划设计 (PBS 扫描)
- 计划优化 (质子特有优化算法)
- 计划评估
- 计划QA

## 质子特有功能
- PBS (Pencil Beam Scanning) 笔形束扫描
- 能量层视图
- Range Shifter 配置
- LET (线性能量传递) 分析
- 鲁棒性评估

## 使用方法
直接打开 `index.html` 即可使用质子治疗计划系统。

## 依赖
- 共享组件库：`../shared/components/`
- 共享样式：`../shared/styles/`
- 共享资源：`../shared/assets/`

## 说明
本系统是放射治疗计划系统的高保真原型示例，用于开发参考。
