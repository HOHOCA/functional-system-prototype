## 修复的问题

### 问题1：navigateTo函数参数错误

**问题描述：**
- `navigateTo` 函数中使用了 `event.currentTarget`，但 `event` 对象未作为参数传递
- 在 `onclick` 属性中使用时，`event` 对象可能不可用或行为不一致

**修复方案：**
```javascript
// 修复前
function navigateTo(url) {
    if (event.currentTarget.classList.contains('disabled')) {
        return;
    }
    window.location.href = url;
}

// 修复后
function navigateTo(url, element) {
    if (element && element.classList.contains('disabled')) {
        return;
    }
    window.location.href = url;
}
```

**修复位置：**
- 更新了所有 `onclick` 属性：`onclick="navigateTo('url', this)"`

**修复效果：**
- ✅ 点击功能正常工作
- ✅ 键盘快捷键功能正常工作
- ✅ disabled 状态检查正常工作
